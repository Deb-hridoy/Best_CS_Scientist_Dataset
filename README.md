# Best_CS_Scientist_Dataset

## Build from Scratch
1. Clone the repo
```bash
git clone https://github.com/Deb-hridoy/Best_CS_Scientist_Dataset.git
```
2. Initialize and activate the virtual environment
3. Install dependencies
```bash
pip install -r requirements.txt
```
