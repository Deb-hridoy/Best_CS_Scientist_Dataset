# Best_CS_Scientist_Dataset

## Problem Statement
The goal of this project is to obtain information on the top 1000 computer scientists (2025-2026) from [this website](https://research.com/scientists-rankings/computer-science). Then the dataset is used to visualize the trends and correlation using a Tableau dashboard. <br/>

Dashboard in Tableau: [Link](https://public.tableau.com/app/profile/hridoy.mahin/viz/DemographicsofComputerScientistsin2025-2026/Dashboard1) <br/>

## Build from Scratch
1. Clone the repo
```bash
git clone https://github.com/Deb-hridoy/Best_CS_Scientist_Dataset.git
```
2. Initialize and activate the virtual environment
3. Install dependencies
```bash
pip install -r requirements.txt
```
